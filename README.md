# Function to Code

*func2code* can get the code of function.

## Import

### ESM

```ts
import func2code from 'func2code';
```

### CJS

```js
const func2code = require('func2code');
```

### Browser

```html
<script src="func2code/index.js"></script>
<script>
  /// <reference path="func2code/browser.d.ts" />
  console.log(func2code);
</script>
```
